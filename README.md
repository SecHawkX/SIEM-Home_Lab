# SIEM-Home_Lab
A complete Mini SIEM Home Lab built with Splunk Enterprise, pfSense, Windows, and Kali Linux for centralized log collection, monitoring, and threat detection.

## Overview

This project demonstrates the design and implementation of a Security Information and Event Management (SIEM) home lab that simulates a real-world Security Operations Center (SOC). The environment centralizes logs from Windows, Kali Linux, and pfSense into Splunk Enterprise for security monitoring, log analysis, and threat detection.

## Objectives

* Build a SOC home lab from scratch
* Centralize logs from multiple endpoints
* Monitor network and endpoint activity
* Perform basic threat hunting
* Learn enterprise SIEM workflows
